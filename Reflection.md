What I have learned from “.net maui” so far is that it has similarities to other coding languages like html and css. 
Despite these similarities, “.net maui” also has quite a bit of restrictions. 
For example, images are more difficult to manage in this language because it seems you will have to somehow scale an images size outside the code to get a size you want. You also can’t use borders in this coding language which is kind of frustrating for me because I like to use them to see how big a section is or to make images look more stylish. 
Another challenge for me was remembering how routing between pages worked. Overall, it wasn’t too difficult to complete the tasks.
